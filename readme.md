# current-path-cli

> Get the path of the current directory in Finder *(macOS)* and Explorer *(Windows)*


## Install

```
$ npm install --global current-path-cli
```


## Usage

```
$ current-path --help

  Example
    $ current-path
    /Users/sindresorhus/dev/current-path
```
